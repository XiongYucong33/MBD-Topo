# Drug-Target Interaction Prediction

A deep learning project for **drug-target interaction (DTI) prediction**.

This repository includes:

- model definition code
- 5-fold cross-validation training script
- multi-fold evaluation script
- high-confidence prediction screening script
- sample training and test data

The project uses:

- **SMILES sequences**
- **protein sequences**
- **molecular graph representations**
- **attention-based drug-protein interaction modeling**
